1995 PHP 1.0: 
	Personal Home Page
	Funcionalidades simples para um site básico
	Não tinha características de linguagem em si

1997 PHP 2.0:
	Oficialmente nomeado PHP/FI 2.0
	Linguagem satandalone
	Recursos limitados

1998 PHP 3.0:
	Primeira versão colaborativa
	Características mais relacionadas a uma linguagem
	Extension API

2000 PHP 4.0:
	Zend Engine 1.0
	Melhoria de perfomance
	Aumenta da modularização
	Uso de super globais: $_GET, $_POST, $_SESSION

2004 PHP 5.0: Mais popular; mais que deu problema
	Zend Engine 2.0
	Primeira versão com Orientação a Objetos
	PHP Data Objects (PDO)
	Operador de exponenciação
	Suporte a JSON
	Namespaces, closures, garbage collection, exceptions handlling
	Novo depurador (phpdbg)

2005 PHP 6.0: Nunca teve
	Aumento da segurança e performance
	Suporte nativo a Unicode
	Até existiram algmas versões
	Nunca recebeu uma release oficial

2015 PHP 7.0:
	Zend Engine 3.0
	Performance até 9x maior
	Escape de códigos Unicode
	Operador null coalescing
	Declaração escalar
	Operador spacceship <=>
	Classes anônimas
	Excessões mais modernas
	Retorno de void
	Libsodium ---> Criptografia de dados
	Foreign function interface

2020 PHP 8.0:
	Zen Engine 4.0
	Just-in-Time compilation
	Locale-independent number convertions
	Named arguments
	Expressão match
	Operador nullsafe
	Enumerations
	Readonly properties
	Fibers
